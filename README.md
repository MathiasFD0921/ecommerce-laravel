# TechNext - E-commerce Minimalista
Tienda virtual con estética inspirada en Apple, desarrollada con **Laravel 11**, **Tailwind CSS** y **SQLite**.
## Características
* **Catálogo Dinámico:** Productos cargados desde base de datos con Eloquent ORM.
* **Carrito de Compras:** Sistema funcional mediante sesiones de Laravel.
* **Diseño Apple Style:** Interfaz limpia con efectos de desenfoque y bordes redondeados.
## Tecnologías
* **Backend:** PHP 8.2 / Laravel 11.
* **Frontend:** Blade & Tailwind CSS.
* **Base de Datos:** SQLite.
## Instalación Rápida
1. Clonar: `git clone https://github.com/tu-usuario/nombre-repo.git`
2. Instalar: `composer install`
3. Configurar: `cp .env.example .env` y `php artisan key:generate`
4. Base de datos: `php artisan migrate:fresh --seed`
5. Ejecutar: `php artisan serve`
